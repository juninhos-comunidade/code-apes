# RFCs — Merge Quest

RFCs são propostas abertas a discussão. ADRs registram decisões aceitas.

## Fluxo

1. Copiar [`TEMPLATE.md`](TEMPLATE.md) → `RFC-NNNN-slug.md`.
2. Abrir discussão (PR ou Linear com label `type:research`).
3. Challenger pode atacar a RFC antes do aceite.
4. Aceite → promover a ADR (ou rejeitar / arquivar).
