## Como propor nova ADR

1. Copiar o formato de uma ADR existente.
2. Ou abrir RFC em `docs/rfc/` se ainda for discussão aberta.
3. Após aceite, linkar no índice e em `docs/architecture/14-decisions-and-open-questions.md`.

Precedência: ver `README-FIRST.md` — ADRs canônicas vencem `sources/` e `docs/history/`.
