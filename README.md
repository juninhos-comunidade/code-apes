# code-apes
A pessoa cola a vaga que ela quer, e nosso sistema transforma aquela descrição numa dungeon pra atravessar. É um RPG dungeon crawler de verdade, andar por andar, sala por sala, turno por turno, só que desenhado com a cara de HQ old school tipo Scott Pilgrim, pixel art vibrante, painel de quadrinho, balão de fala, onomatopeia tipo "POW" e ...
